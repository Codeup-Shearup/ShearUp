Project Description

ShearUp is a full-stack application intended to facilitate a streamlined connection between barbers and clients. Built using MVC design pattern, Spring Boot with Spring Security, Thymeleaf, MySQL as well as integrating Mapbox, MailGun, amd Filestack APIs, ShearUp provides a sleek, sophisticated platform to revitalize an industry deeply impacted by the pandemic of 2020.
